# trade-app-backend（课设精简版）

本仓库用于实现一个二手交易平台后端的**课设精简版 MVP**（1 周可交付、可运行）。

## 目标（本周）

- 用户注册/登录（JWT Access Token）
- 用户资料（不做头像上传）
- 商品发布与浏览（不含图片）
- 最小交易闭环（下单/取消/完成；不接支付、不做物流）
- PostgreSQL + 迁移 + Docker 本地运行

## 非目标（本周不做）

- 举报/后台审核/收藏/聊天/通知/评价
- 头像/商品图片上传、对象存储（MinIO/S3）
- Redis（缓存/会话/限流/队列）
